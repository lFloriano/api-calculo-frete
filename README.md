# **[Loja Virtual] - Desafio Tecnico: API RESTful para Cálculo de Frete **

### **Autor**
- Lucas Floriano

## **Proposta do Projeto**

O projeto consiste em:
- **API RESTful:** CRUD de pedidos e cálculo de frete.
- **Acesso a Dados:** Implementação de acesso a dados através de In-Memory repository.

  ## **Tecnologias Utilizadas**

- **Linguagem de Programação:** C#
- **Frameworks:**
  - ASP.NET Core Web API
  - .NET 9
  - AutoMapper
  - Swagger
  - xUnit
  
- **Banco de Dados:** Acesso a dados através de In-Memory repository.

## **Funcionalidades Implementadas**
- **Testes Unitários com xUnit:** Construção da aplicação aplicando TDD na camada de domínio.
- **CRUD de Pedidos:** Permite criar, editar, visualizar e excluir Pedidos.
- **Cálculo de Frete:** Permite calcular frete de pedidos.
- **API RESTful:** Exposição de endpoints para operações CRUD via API.
- **Documentação da API:** Documentação dos endpoints da API utilizando Swagger.
- **Seed de Pedidos:** Criação de pedidos automática ao executar a aplicação para facilitar testes dos endpoints.

  ## **Observações**
- Este projeto tem finalidade acadêmica e não aceita contribuições externas.
